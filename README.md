# 🛍️ Thrift & Handmade Store ER Diagram

This repository contains the ER diagram for a small Instagram-based business selling thrifted and handmade products.

## 📌 Features Covered

* Product management (thrift & handmade)
* Product variants (size, color)
* Inventory tracking
* Customer management
* Order handling
* Payment tracking
* Shipping details

## 🧠 Design Highlights

* Used **Product Variants** to support size & color combinations
* Implemented **specialization** for:

  * Thrift Products (unique items)
  * Handmade Products (multi-quantity items)
* Maintained proper relationships between:

  * Customers → Orders
  * Orders → Items
  * Products → Variants
